# Configs

## Usage
Create the following files at the project root.
### Typescript
> `tsconfig.json`

```json
{
  "extends": "@beastly-brawl-showdown/configs/tsconfig"
}
```

### EsLint (Under Construction)
> `eslint.config.js`

```js
import config from '@beastly-brawl-showdown/configs/eslint';
export default config;
```
